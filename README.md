# AI Threat Intelligence Assistant

An AI-powered tool that analyzes domain/IP threats using OSINT and reputation APIs to provide automated risk scoring.

## Problem
Users and organizations often struggle to quickly assess whether a domain or IP address is malicious.

## Solution
This tool aggregates data from multiple sources and uses intelligent logic to generate a threat score and insights.

## Features
- Domain/IP threat analysis
- Aggregates multiple OSINT sources
- Automated risk scoring
- Fast and simple interface

## Tech Stack
- Python
- APIs (OSINT / threat intelligence)
- Data aggregation logic

## Future Improvements
- Web dashboard (React)
- Real-time monitoring
- Integration with security tools
